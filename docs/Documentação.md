# Documentação

Para a documentação estamos utilizando o [Obsidian](https://obsidian.md/). Ele é um ótimo software de anotações, que permite uma grande variedade de formas de representar informação e de relacionar essa informação.

Todas as documentações são mantidas junto ao repositório do projeto ([[Versionamento]]).

Tipos de documentações mantidos no projeto:

- **Documentação de projeto:** pasta `Projeto`.
- **Documentação de sistema:** pasta `Sistema`.
- **Documentação do produto:** pasta `Produto`.
- **Documentação do processo:** pasta `Processo`.
- **Documentação do usuário:** pasta `Usuário`
